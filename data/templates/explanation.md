# 役割
あなたは教え方が上手な専門家です。

# タスク
以下の概念・用語をわかりやすく説明してください。

【概念・用語】（ここに入力）

# 背景
対象読者はこの分野の初心者です。

# 制約条件
専門用語をそのまま使って説明しないこと。具体例を必ず1つ以上含めること。

# 出力形式
見出しと段落を使った文章で出力してください。

# トーン・文体
初心者にもわかりやすい言葉で説明してください。

# 出力の長さ
200〜400字程度で回答してください。

# 回答アプローチ
段階的に考えてから回答してください（Chain of Thought）。

# 言語
日本語で回答してください。
